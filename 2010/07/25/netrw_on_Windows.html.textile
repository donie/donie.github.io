<h4> <a href="/2010/07/25/netrw_on_Windows.html"> vim/netrw </a> </h4>
For Windows，only。 真羡慕 Mac 和 *nix 上的人，不用烦劳这些稀奇古怪的问题。

最好升级一下 netrw 包，at here： 
<a href="http://mysite.verizon.net/astronaut/vim/index.html#NETRW" rel="external">http://mysite.verizon.net/astronaut/vim/index.html#NETRW</a>

在 vimrc 中加上这些：

pre. let g:netrw_cygwin = 0
let g:netrw_scp_cmd = “C:\\path\\to\\putty's\\scp\\pscp.exe -pw yourpasswd”
let g:netrw_sftp_cmd = “C:\\path\\to\\putty's\\sftp\\pftp.exe -pw yourpasswd”
let g:netrw_list_cmd = “plink username@hostname -pw yourpasswd ls -Fa

pre. :e scp://user@hostname//path/

path 之前必须多加上一个 /

Voila。（厄，被 Howard 带的。）
